javaScript工具jutils

Created2021-02-23|Updated2021-02-23|前端
|Word count:513|Reading time:2min|Post View:

JavaScript常用函数库

在日常工作生活中，会经常用到一些日期格式化，url相关操作，浏览器类型判断，常用验证格式等等函数，虽然大部分只需谷歌/百度一下就能找到，但是大多数都存在着一些问题，于是整理了网上和自己平常用到的工具类，方便大家以后的使用，提升开发效率。

Git地址：https://github.com/dong-sir/jutils

安装

直接用 <script> 引入

直接下载并用 <script> 标签引入，jutils 会被注册为一个全局变量。

CDN

你也可以这样使用最新版本：

NPM

在 jutils 应用于大型项目时推荐使用 NPM 安装。NPM 能很好地和 webpack 模块打包器配合使用。

交流群（QQ）

  • 709060365

API 目录

浏览器

  • getBrowserInfo 获取浏览器信息
  • isCss3Support 判断浏览器是否支持css3

数组

  • arrayUnique 数组去重
  • arrayEqual 判断两个数组是否相等

日期

  • formatDate 时间戳的转换（自定义格式）
  • getTimeInterval 获取两个时间的间隔的天、小时、分钟和秒

API 说明

getBrowserInfo

获取浏览器信息

isCss3Support

判断是否支持css3

arrayUnique

数组去重

arrayEqual

判断两个数组是否相等

formatDate

时间戳的转换（自定义格式）

年、月、日、时、分、秒

getTimeInterval

获取两个时间的间隔，返回间隔的天、小时、分钟和秒。 注意：结束时间要大于开始时间否则返回空
